# Project: Linux & Docker Infrastructure Automation

## Overview
A core automation project focusing on system task scheduling and containerized web delivery. This project demonstrates foundational DevOps skills in managing Linux environments and deploying immutable Nginx web servers.

## 🛠️ Tech Stack
* **OS**: Linux (Debian-based)
* **Automation**: Cron Jobs for scheduled system tasks
* **Containerization**: Docker (Nginx Alpine Image)
* **Networking**: Port Mapping (8080:80)

## 📋 Key Learnings
- **Cron Job Automation**: Configured automated scripts on distributed Nautilus application servers to optimize background tasks.
- **Dockerization**: Built a custom Docker image to wrap a static web asset, ensuring an immutable and portable environment.
- **Deployment**: Successfully mapped local ports to container services, verifying connectivity in a live environment.
